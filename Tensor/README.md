# Tensor-document-ZH

### Tensor 类描述

* `Tensor`类为`MINI-MLsys`中的张量类，支持维度：`1-dim、2-dim、3-dim`

  实现的时候认为张量都是3维的

* `Tensor`类通过`Armadillo.cube`存储数据

* `Tensor`类提供私有成员变量`std::vector<uint32_t>shape`存储张量格式



### Tensor 类功能及函数
