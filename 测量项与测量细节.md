## 算子尺寸描述

| 算子           | 各维度解释（H=2048, D=128, NH=16, NKV=4, T=4, Tk=256, I=6144, V=59264） |
| -------------- | ------------------------------------------------------------ |
| GET_ROWS       | 输入1 `W{H,V,1,1}`= `{2048,59264,1,1}` 词嵌入矩阵；输入2 `ids{T,1,1,1}`= `{4,1,1,1}` token 序列；输出 `{H,T,1,1}`= `{2048,4,1,1}`。 |
| RMS_NORM       | 输入 `x{H,T,1,1}`= `{2048,4,1,1}`；输出形状不变 `{2048,4,1,1}`。按通道归一。 |
| MUL            | 输入 `x{H,T,1,1}`= `{2048,4,1,1}` 与缩放 `γ{H,1,1,1}`= `{2048,1,1,1}`；输出 `{2048,4,1,1}`。逐元素乘，γ按 T 广播。 |
| MUL_MAT        | 例1（H→H 投影）：`W{H,H,1,1}`= `{2048,2048,1,1}` · `X{H,T,1,1}`= `{2048,4,1,1}` → `Y{H,T,1,1}`= `{2048,4,1,1}`。 |
| RESHAPE        | 将 `{H,T,1,1}`= `{2048,4,1,1}` 重塑为 `{D,NH,T,1}`= `{128,16,4,1}`，即把隐藏维分成头维与每头维。元素总数不变。 |
| ROPE           | 对 `Q/K{D,NH,T,1}`= `{128,16,4,1}` 施加旋转位置编码；辅助张量给步长/位置 `{T,1,1,1}`= `{4,1,1,1}`；输出形状不变 `{128,16,4,1}`。 |
| VIEW           | 仅视图重解释，不拷贝。此处将 `{D,T,group,1}`= `{128,4,4,1}` 视为 `{D×group,T,1,1}`= `{512,4,1,1}`，便于按行写入缓存；这里 `group=NKV=4`。 |
| SET_ROWS       | 将当前步 `src{D×group,T,1,1}`= `{512,4,1,1}` 依据 `ids{T,1,1,1}`= `{4,1,1,1}` 写入缓存 `dst{D×group,L,1,1}`= `{512,4096,1,1}`；`L` 为实现层面的“缓存行数”（此处 4096，常等于 `Tk×某分组数`）。 |
| PERMUTE        | 轴交换以匹配后续算子。例如 `{D,NH,T,1}`= `{128,16,4,1}` → `{D,T,NH,1}`= `{128,4,16,1}`。形状元素数不变。 |
| CPY            | 拷贝或类型/设备转换，形状不变。例：`{256,64,1,1}` → `{256,64,1,1}`。 |
| FLASH_ATTN_EXT | 融合注意力：`Q{D,T,NH,1}`= `{128,4,16,1}` 与 `K/V{D,Tk,NKV,1}`= `{128,256,4,1}`；输出 `ctx{D,NH,T,1}`= `{128,16,4,1}`。内部含缩放、掩码、softmax 与乘 V。 |
| ADD            | 残差相加：`a{H,T,1,1}` + `b{H,T,1,1}`= `{2048,4,1,1}` → `{2048,4,1,1}`。 |
| SWIGLU         | 典型实现：线性得到 `2I` 并拆成 `up{I,T,1,1}` 与 `gate{I,T,1,1}`，输出 `{I,T,1,1}`。此处等价地展示为 `{12288,4,1,1}` → `{6144,4,1,1}`，其中 `2I=12288`、`I=6144`。 |

| 符号 | 含义                                                   |
| ---- | ------------------------------------------------------ |
| H    | 隐藏维度（每个 token 的通道数，embedding/层输出维）    |
| D    | 每个注意力头的维度（per-head dim）                     |
| NH   | 注意力的 Q 头数（query heads）                         |
| NKV  | K/V 头数（用于 GQA；多头=MHA 时 NKV=NH，MQA 时 NKV=1） |
| T    | 本次前向计算的 token 数（当前步序列长度或微批大小）    |
| Tk   | 已缓存的历史长度（KV-cache 中的过去 token 数）         |
| I    | FFN 中间维度（如 SwiGLU 的 up/gate 维）                |
| V    | 词表大小（vocabulary size）                            |

## 测试尺寸约定

| 算子                     | 需要覆盖的维度及取值集合（用于做笛卡尔组合）                 |
| ------------------------ | ------------------------------------------------------------ |
| GET_ROWS                 | H∈{1024,2048,4096}；V∈{32768,65536,131072}；**T∈{4,64,256,1024}** |
| RMS_NORM                 | H∈{1024,2048,4096}；**T∈{4,64,256,1024}**                    |
| MUL（通道缩放）          | H∈{1024,2048,4096}；**T∈{4,64,256,1024}**（同时测 γ 形状 {H,1,1,1} 与 {H,T,1,1}） |
| ADD（残差）              | H∈{1024,2048,4096}；**T∈{4,64,256,1024}**（测广播 {H,1,1,1}+{H,T,1,1} 与等形相加） |
| SCALE                    | H∈{1024,2048,4096}；**T∈{4,64,256,1024}**                    |
| MUL_MAT：H→H 投影        | H∈{1024,2048,4096}；**T∈{4,64,256,1024}**（W{H,H}·X{H,T}→{H,T}） |
| MUL_MAT：FFN up/gate     | H∈{1024,2048,4096}；I∈{4096,6144,8192,12288}；**T∈{4,64,256,1024}**（W{I,H}·X{H,T}→{I,T}） |
| MUL_MAT：FFN down        | H∈{1024,2048,4096}；I∈{4096,6144,8192,12288}；**T∈{4,64,256,1024}**（W{H,I}·X{I,T}→{H,T}） |
| MUL_MAT：K/V 投影（GQA） | 固定 H 与 NH 使 D=H/NH 为整数；NH∈{8,16,32}；NKV∈{1,NH/2,NH}；**T∈{4,64,256,1024}**。示例 W{D·NKV,H}·X{H,T}→{D·NKV,T} |
| RESHAPE                  | `{H,T,1,1}↔{D,NH,T,1}`；H∈{1024,2048,4096}；NH∈{8,16,32}；D=H/NH；**T∈{4,64,256,1024}** |
| ROPE                     | 输入 `{D,NH,T,1}`；H∈{1024,2048,4096}；NH∈{8,16,32}；D=H/NH；**T∈{4,64,256,1024}** |
| VIEW                     | 元素数保持不变的视图重排。用同一组 H、NH、NKV、D 与 **T∈{4,64,256,1024}**、Tk∈{0,256,2048,8192} 生成多种视图 |
| SET_ROWS（写 KV）        | D、NKV、Tk、T：H∈{1024,2048,4096}；NH∈{8,16,32}；D=H/NH；NKV∈{1,NH/2,NH}；Tk∈{0,256,2048,8192}；**T∈{4,64,256,1024}** |
| PERMUTE                  | 输入 `{D,NH,T,1}`；H∈{1024,2048,4096}；NH∈{8,16,32}；D=H/NH；**T∈{4,64,256,1024}** |
| CPY                      | 以元素数 E 为横轴：E∈{1M,4M,16M,64M,256M}（如 {2048,512,1,1} 等）。与 T 无关 |
| FLASH_ATTN_EXT           | Q{D,NH,**T**}, K/V{D,Tk,NKV}：H∈{1024,2048,4096}；NH∈{8,16,32}；D=H/NH；NKV∈{1,NH/2,NH}；**T∈{4,64,256,1024}**；Tk∈{0,256,2048,8192} |
| SWIGLU                   | I∈{4096,6144,8192,12288}；**T∈{4,64,256,1024}**（输入 up{I,T} 与 gate{I,T}） |
| GELU                     | I∈{4096,6144,8192,12288}；**T∈{4,64,256,1024}**              |
