# 能耗测量实验

## 分支管理方法
* 主分支：main，用于更新实验数据和实验模板
* 各个框架分别测量：拉取main分支到本地，创建新分支（名称为：llamacpp,qnn,mnn），每个分支独立负责一个推理框架的测试数据，提交pr合并到main分支
* 实验合作同学参与开发方法：拉取对应框架分支到本地，创建新分支，如：（llamacp_alice），完成开发后向所述框架分支提交pr，如（llamacpp），不可以直接提交pr到main分支

## 文件说明
* ops_time为第一个实验，具体描述见**测量项与测量细节.md**
* 实验数据保存位置：ops_time下每一个算子给出了一个输出csv文件的模板result.csv,每个框架在该文件夹下独立保存,如：result_llama.csv
* src用于保存每个框架实验所需的源码，如果测试脚本需要用到框架源码，clone的时候不要将git文件放入，以免仓库过大
